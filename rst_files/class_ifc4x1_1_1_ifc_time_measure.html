Definition from ISO/CD 10303-41:1992: A time measure is the value of the duration of periods. Measured in seconds (s) or days (d) or other units of time. Type: REAL

NOTE Corresponding ISO 10303 name: time_measure, please refer to ISO/IS 10303-41 for the final definition of the formal standard.
